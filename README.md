# LogAnalyzer
[![Build Status](https://travis-ci.org/lysevi/logan.svg?branch=master)](https://travis-ci.org/lysevi/logan)

# Features
* Log viewing.
* Auto scrolling.
* Reload logs by timer.
* Highlight by regex.
* Filtration by regex.
* Filtration by time.
